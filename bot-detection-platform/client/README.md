# Bot Detection Platform - Client

React frontend for the bot detection platform.

## Development

```bash
npm run dev
```

Runs on `http://localhost:3000`

## Build

```bash
npm run build
```

## Testing

```bash
npm test
```

## Features (Phase 3+)

- Website pages (Home, Products, Blog, etc.)
- User authentication
- Admin dashboard
- Session analysis
- Bot detection results
